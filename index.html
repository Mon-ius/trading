Strategic Communication, Experience, and Bubbles in Experimental Asset Markets

A Unified Computational Framework for LLM Agent Trading with Sobel Communication

Motivation

Asset price bubbles remain one of the most persistent and damaging phenomena in financial markets. Laboratory experiments (Smith et al. 1988) consistently reproduce bubble-crash patterns even when fundamental value is common knowledge. Three open questions motivate this work:

  • Q1. Can computational agents — including LLMs — replicate the bubble dynamics observed in human experiments, or do they trade more rationally?
  • Q2. How does trader experience affect bubble magnitude? Dufwenberg et al. (2005, AER) showed experience dampens bubbles, but the mechanism is not fully understood.
  • Q3. What is the role of strategic communication? When agents can lie and deceive (Sobel 2020, JPE), does information revelation improve or deteriorate market efficiency?

These questions require a unified framework that integrates market microstructure, agent heterogeneity, game-theoretic communication, and experience effects — which no existing simulator provides.

Our Contribution

We present an interactive experimental platform that merges four foundational research paradigms into a single, reproducible simulator:

CARA Utility Agents

Heterogeneous risk preferences and private valuations drive endogenous price formation via continuous double auction (Lopez-Lira 2025).

Stochastic + LLM Hybrid

Stochastic CARA model by default; optional LLM API integration replaces decision-making with real AI agent reasoning.

Sobel Communication

Formal lying/deception model with ON/OFF toggle enables controlled ablation studies of strategic talk (Sobel 2020 JPE).

Experience Sessions

Configurable experience replication validates the Dufwenberg (2005 AER) finding that bubbles diminish with repetition.

Theoretical Foundations

Four Papers, One Unified Framework

Lopez-Lira (2025) — “Can Large Language Models Trade?”

Open-source framework for LLM trading agents with persistent order book, limit orders, partial fills, dividends, and heterogeneous strategies. Demonstrates LLMs can function as value investors, momentum traders, and market makers. Provides the CARA utility agent architecture and CDA matching engine we adopt.

U(W) = −exp(−γW) / γ   |   bidi = ψi − γiσ²/2   |   aski = ψi + γiσ²/2

Dufwenberg, Lindqvist & Moore (2005, AER) — “Bubbles and Experience”

Laboratory CDA markets with 6 traders over 4 sessions of 10 periods. Key result: bubble-crash patterns diminish with experience. Even ⅓ experienced traders suppress bubbles in mixed markets. Defines the bubble metrics (Haessel-R², NAPD, amplitude, turnover) we use for quantitative evaluation.

α* ≈ ⅓ — min experienced fraction to suppress bubbles  |  R² = 1 − Σ(P−FV)²/Σ(FV−F̄V)²

Henning et al. (2025) — “LLM Trading: LLM Agent Behavior in Experimental Asset Markets”

Tests 6 LLM models (Claude, GPT-4o, Grok-2, Mistral, GPT-3.5, Gemini) in single-model and Battle Royale markets. LLMs trade more rationally than humans. Claude achieves MSE=0.54 (Rational hypothesis). Provides the constant FV model (E[div]/r) and R/H/E hypothesis classification framework.

FV = E[div] / r = (dL + dH) / 2r = (0.4 + 1.0) / (2 × 0.05) = 14

Sobel (2020, JPE) — “Lying and Deception in Games”

Formal sender-receiver framework using Austin’s (1975) speech-act theory. Distinguishes lying (locution: m = mθ₀, θ ∉ Θ₀) from deception (illocution: inducing inferior beliefs) and damage (perlocution: welfare loss). Key insight: lies need not be deceptive (Remark 7); deception is possible without lying (Example 3).

Lie (Def 1): m = mθ₀, θ ∉ Θ₀  |  Deception (Def 4): μ(·|m) = pμ(·|n) + (1−p)ρ  |  Credulity (Def 6)

Methodology: Unified Framework Architecture

Two-Phase Experiment with Stochastic/LLM Agent Hybrid

Experiment Protocol

  1. Agent Creation: n agents with CARA utility (γi), private valuations (ψi), heterogeneous endowments (cash, shares). If API key provided, agents use LLM reasoning; otherwise, CARA stochastic model.
  2. Phase 1 — Silent CDA: T rounds of dividend payment → interest accrual → order submission (reservation prices) → CDA matching. No communication. Establishes baseline.
  3. Phase 2 — Sobel Communication: Same agents, positions carry over. Each round adds pre-trade Sobel sender-receiver phase. Receivers update effective ψ based on credulity. Toggle OFF = silent control.
  4. Experience Sessions: 0–5 additional silent sessions with reset endowments but persistent experience. Tests Dufwenberg (2005) experience effect.

Stochastic vs LLM Decision-Making

Stochastic: bid = ψ − γσ²/2 + noise
LLM: bid = LLM(context, history, ψ, γ)

The same experiment protocol runs identically. Only the decision function changes. This enables direct comparison of stochastic and AI agent behavior.

Methodology: Agent Design & Market Microstructure

CARA Utility, Reservation Prices, and CDA Matching

Agent Parameterization

Each agent i is fully characterized by four components:

ParameterSymbolDistribution
Risk aversionγiRisk-loving: γ∈[−0.01,−0.001]; Neutral: γ∈[0.01,0.03]; Averse: γ∈[0.05,0.12]
Private valuationψiUniform[base−spread, base+spread]
Initial cashci,0Normal(μcash, σ2endow)
Initial sharessi,0Poisson(μshares) with variance

CARA Reservation Prices

bidi = ψi − γi · σ² / 2
aski = ψi + γi · σ² / 2

Risk-loving agents (γ<0): bid above ψ (aggressive buyer). Risk-averse agents (γ>0): ask above ψ (demand premium to sell). This heterogeneity generates trade volume and potential mispricings.

CDA Matching Engine

  1. Order Collection: Each agent submits one bid and one ask based on CARA-optimal reservation prices
  2. Sorting: Bids sorted descending; asks sorted ascending
  3. Matching: While max(bid) ≥ min(ask), execute trade at midpoint price: P = (bid + ask) / 2
  4. Validation: Buyer must have sufficient cash; seller must hold shares. No short-selling, no margin.
  5. Settlement: Atomic transfer of cash and shares between counterparties

Per-Round Dynamics

divt ∼ {dL, dH} with Pr = 0.5 (Henning 2025)
cashi,t+1 = cashi,t × (1 + r) + divt × sharesi,t
VWAPt = Σ(Pk × Vk) / Σ Vk

Each round: (1) pay stochastic dividend, (2) accrue interest on cash, (3) optional Sobel communication, (4) compute orders, (5) CDA match, (6) record metrics.

Risk Composition Effects

The simulator reveals how agent population composition drives bubble formation:

  • High Risk-Loving (RL > 50%): Narrow spreads, aggressive bidding → prices overshoot FV → large NAPD and amplitude. Classification trends toward H-type.
  • Balanced (RL≈RN≈RA): Moderate spreads, prices oscillate around FV → R-type classification in most seeds.
  • High Risk-Averse (RA > 50%): Wide spreads, conservative bidding → prices undershoot FV → negative deviations, low turnover.

Comparison: Stochastic vs LLM Agents

When an API key is provided, LLM agents replace the stochastic CARA model. Henning (2025) found that top-tier LLMs (Claude, GPT-4o) achieve R-type behavior with MSE < 1, while weaker models (GPT-3.5, Grok-2) produce H-type bubble dynamics. Our unified framework enables direct A/B comparison on the same experiment protocol.

Main Result 2: Strategic Communication & Deception

Sobel (2020 JPE) Model Applied to CDA Markets

Sobel’s Three-Level Framework

LevelConceptDefinitionTrading Implementation
LocutionLying (Def 1)m = mθ₀, θ ∉ Θ₀Agent reports ψ̂ ≠ ψtrue
IllocutionDeception (Def 4)μ(·|m) farther from truthReport shifts others’ beliefs away from true ψ
PerlocutionDamage (Sec V)Worse receiver decisionOthers trade at worse prices due to false signal

Bluffing by Risk Type (Sec VI)

Buyers (want low price): reported = ψ × (1 − scale × dampen)
Sellers (want high price): reported = ψ × (1 + scale × dampen)

Risk-loving agents bluff more aggressively (1.5× amplifier). Experienced agents dampen bluffing: dampen = max(0.3, 1 − exp × 0.15).

Receiver Credulity (Def 6)

effectivej = ψj(1−w) + avg(signals) × w
w = credulity / (skepticism[riskType] × (1 + exp × 0.1))

Key Finding: Communication & Efficiency

The communication ON/OFF toggle reveals the causal effect of strategic talk:

  • Low deception strength + moderate credulity: Communication improves efficiency. Agents reveal partial information about ψi, enabling better matching.
  • High deception strength + high credulity: Communication harms efficiency. Credulous agents act on lies, creating misallocation.
  • High deception + low credulity: Lies are discounted; communication has negligible effect (Sobel Remark 7: lies need not be deceptive).

All metrics are tracked per round, per risk type, and across phases — enabling fine-grained analysis of who lies, who is deceived, and who benefits.

Our Computational Replication

The experiment includes configurable experience sessions (0–5 additional rounds). After each session, agents increment their experience counter, which affects two channels:

Channel 1: Reduced Deception Aggressiveness

dampen = max(0.3, 1 − experience × 0.15)

Experienced agents bluff less — they learn that aggressive lying doesn’t improve outcomes. At experience=4, dampen=0.4 (60% reduction from naive level).

Channel 2: Increased Skepticism

effective_credulity = credulity / (1 + experience × 0.1)

Experienced receivers discount messages more — they learn to ignore strategic signals. This maps to Sobel’s insight that communication equilibria shift when receivers become skeptical.

Expected Outcome

Across sessions, R² should increase and NAPD should decrease monotonically, reproducing the “bubbles and experience” finding. The simulator provides quantitative comparison against Table 2 values.

Additional Results: Allocative Efficiency & Coase Theorem

Welfare Analysis, P&L Distribution, and Visualization Framework

Coase Theorem Test

Regardless of initial endowment allocation, voluntary trade in a CDA should move assets toward agents with the highest private valuation ψi, provided transaction costs are low.

W = Σi ψi × sharesi   (welfare)
Efficiency = (W − Wmin) / (Wmax − Wmin)
Spearman ρ(ψ, shares) → 1 if Coase holds

We test this by plotting ψi vs final share holdings (Fig. 2). A strong positive correlation indicates efficient allocation. Color-coding by risk type reveals whether risk preferences interfere with Coase convergence.

Conclusions & Future Directions

Summary of Contributions and Open Research Questions

Key Contributions

  1. Unified experiment integrating experimental finance (Dufwenberg 2005 AER), LLM agent trading (Lopez-Lira 2025, Henning 2025), and game-theoretic communication (Sobel 2020 JPE) into a single reproducible platform.
  2. Hybrid agent architecture: stochastic CARA utility model by default, with optional LLM API integration. Same experiment protocol, different decision engine — enabling direct A/B comparison.
  3. Formal Sobel communication model with ON/OFF toggle for controlled ablation of lying, deception, and damage effects on market efficiency.
  4. Computational replication of the Dufwenberg (2005) experience effect, validating that bubble-crash patterns diminish with trader experience in silico.
  5. Open-source, no-build SPA deployable to GitHub Pages. Fully interactive: adjustable parameters, real-time charts, Trading Floor canvas animation, and export (JSON/CSV).

Main Findings

  • CARA agents with heterogeneous ψi converge toward efficient allocation (Coase theorem generally holds)
  • Sobel communication improves OR hinders efficiency depending on the deception–credulity parameter space
  • Experience robustly reduces bubble magnitude: R² increases and NAPD decreases across sessions
  • Risk-loving agents drive bubble formation; balanced populations produce R-type (Rational) pricing

Future Directions

  • LLM agent communication: Let LLM agents generate natural-language messages (not just price signals) and classify Sobel categories via NLP
  • Algorithmic collusion detection: Test whether LLM agents learn to coordinate prices through repeated interaction (Fish et al. 2024)
  • Regulatory sandboxing: Simulate mandatory disclosure requirements, communication bans, and position limits to evaluate policy interventions
  • Multi-asset markets: Extend to correlated assets with information spillover and contagion dynamics
  • Mechanism design comparison: CDA vs call auction vs frequent batch auction — which mechanism is most robust to LLM strategic behavior?
  • Human-AI hybrid markets: Mixed populations of human subjects and AI agents to study emergent interaction dynamics

CARA Utility (Lopez-Lira)

Constant Absolute Risk Aversion. Parameter γ determines risk attitude: γ > 0 risk-averse, γ = 0 neutral, γ < 0 risk-loving.

U(W) = −exp(−γ · W) / γ
Lopez-Lira (2025) §2; DLM (2005) fn.5

Fundamental Value (Henning)

Constant FV derived from perpetuity formula with stochastic dividends and fixed interest rate.

FV = E[div] / r = (dL+dH) / 2r
Henning (2025) Eq.1; DLM (2005) §I fn.5

Reservation Prices

CARA-optimal bid/ask prices incorporating risk premium from market uncertainty σ.

bid = ψ − γσ²/2  |  ask = ψ + γσ²/2
Lopez-Lira (2025) §2.2

Sobel Lying (Def 1)

Message m is a lie given θ if m has accepted meaning mθ₀ and θ ∉ Θ₀. Speaker believes the report false.

Lie: m = mθ₀, θ ∉ Θ₀
Sobel (2020 JPE) §III.A Def 1, p.912

Sobel Deception (Def 4)

Message m is deceptive if beliefs induced by m are farther from truth than beliefs induced by alternative n.

μ(·|m) = pμ(·|n) + (1−p)ρ
Sobel (2020 JPE) §IV.A Def 4, p.920

α* Experience Threshold

Critical experienced fraction to suppress bubbles. R² increases and NAPD decreases with experience sessions.

α* = min{α : NAPD(α) < ε}
DLM (2005) §II Table 2 p.1735
