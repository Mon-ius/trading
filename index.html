Bubbles & Experience with LLM Trading Agents

Replacing the human laboratory of DLM (2005) with Lopez-Lira (2025) CARA / LLM agents

Dufwenberg, Lindqvist & Moore (2005, AER) • Lopez-Lira (2025)

Interactive Virtual Trading Simulator

Motivation & Contribution

DLM (2005, AER) showed in human laboratories that bubbles in CDA markets diminish with experience. We replace those human subjects with computational agents in the spirit of Lopez-Lira (2025). Two questions:

  • Q1. Can stochastic CARA / LLM agents reproduce the DLM declining-FV bubble-and-crash pattern?
  • Q2. Does the DLM experience effect — bubbles shrink across replays of the same market with the same agents — survive when humans are replaced by AI agents?

DLM (2005, AER) asset

Declining FV(t) = (T−t)×E[d]; stochastic dividends ∈ {0, 2·E[d]}; CDA market.

Lopez-Lira (2025) agents

CARA utility, heterogeneous γ, reservation prices bid/ask = belief ∓ γσ²/2.

Experience replay

Same agents, reset endowments, persistent experience counter. Each replay halves residual bias.

Optional LLM agents

Provide an API key (Claude / GPT) and the same protocol runs with real LLM decisions.

Theoretical Foundations

Two Papers, One Unified Framework

Dufwenberg, Lindqvist & Moore (2005, AER) — “Bubbles and Experience”

Laboratory CDA markets with 6 human traders over multiple sessions. The asset has finite life T; each period it pays a stochastic dividend ∈ {0, 2·E[d]}, so the rational FV declines linearly: FV(t) = (T−t)×E[d]. Main result: bubble-and-crash patterns diminish when the same subjects replay the market (experience effect). Transfer treatment: just one third experienced traders suffice to suppress bubbles in mixed markets. We adopt their bubble metrics (Haessel-R², NAPD, amplitude, turnover) verbatim.

FV(t) = (T − t) × E[d]  |  divt ∈ {0, 2·E[d]}  |  R² = 1 − Σ(P−FV)²/Σ(FV−F̄V)²

Lopez-Lira (2025) — “Can Large Language Models Trade?”

Open-source framework for LLM trading agents with a persistent order book, limit orders, partial fills, dividends, and heterogeneous strategies. Demonstrates that LLMs can function as value investors, momentum traders, and market makers. Provides the CARA utility agent architecture, the closed-form reservation prices, and the CDA matching engine that we use as the in-silico replacement for human subjects in DLM (2005).

U(W) = −exp(−γW) / γ  |  bidi = beliefi − γiσ²/2  |  aski = beliefi + γiσ²/2

Unification: bubble experiment × LLM agents

The DLM (2005) declining-FV asset and CDA market structure are kept exactly. The only thing replaced is the human trader: instead of a subject in a z-Tree lab, each seat is filled by a CARA-stochastic agent (or, optionally, by a real LLM call). Belief dynamics follow DLM intuition: experienced agents track FV with small noise; inexperienced agents are anchored, optimistic, and momentum-chasing. Each replay halves residual bias — our minimal computational analogue of "learning from playing the market".

beliefexp = FV·(1+noise·ε)  |  beliefinexp = anchored + biased + momentum, all × 0.5experience

Methodology: Unified Framework Architecture

DLM (2005) Bubble Experiment Driven by Lopez-Lira (2025) Agents

Experiment Protocol

  1. Agent Creation: n agents with CARA risk parameter γi, randomly assigned experience type (fraction α experienced), heterogeneous initial cash and shares.
  2. Belief Update: Each period, experienced agents draw beliefs around FV(t) with small noise; inexperienced agents combine FV anchoring, optimism bias, momentum chasing, and noise.
  3. CDA Round: Each agent submits one bid and one ask at CARA reservation prices. Orders shuffled, matched if bid ≥ ask, executed at midpoint. Dividends paid to share-holders.
  4. Repeat T periods: FV declines linearly to zero. Bubble metrics (Haessel-R², NAPD, amplitude, turnover) computed over the realized price path.
  5. Experience Replays: 0–5 additional sessions with reset endowments but persistent experience. Each replay halves residual inexperienced bias — the DLM "learning from playing" channel.

Stochastic vs LLM Decision-Making

Stochastic: bid = belief − γσ²/2
LLM: belief = LLM(context, history, FV(t), risk type)

The simulation protocol is identical in both modes; only the belief-formation function changes. This enables direct A/B comparison between the closed-form CARA baseline and real LLM agents (Claude / GPT) on exactly the same DLM market.

Methodology: Agent Design & Virtual Trading

CARA Utility, Reservation Prices, and CDA Matching

Agent Parameterization

Each agent i is characterized by:

ParameterSymbolDistribution
Risk aversionγiRL: γ ∈ [−0.008, 0.002]; RN: γ ∈ [0.002, 0.012]; RA: γ ∈ [0.012, 0.060]
Experience typeei{experienced, inexperienced}; experienced fraction = α
Initial cashci,0cash × max(0.3, 1 + σendow·ε)
Initial sharessi,0shares × max(0, 1 + σendow·ε)

Belief Update (per period)

decay = 0.5experience
exp: belief = FV(t)·(1 + 0.05·decay·ε)
inexp: belief = [FV(t)(1−a) + FV(0)·a]·(1+bias) + mom·ΔP + noise

CARA Reservation Prices

bidi = beliefi − γi·σ²/2  |  aski = beliefi + γi·σ²/2

CDA Matching Engine

  1. Order Collection: each agent submits one bid and one ask at the CARA-optimal reservation prices
  2. Shuffle: order list is randomized to remove arrival-order bias
  3. Matching: bids sorted descending, asks sorted ascending. While best-bid ≥ best-ask, execute at midpoint P = (bid+ask)/2
  4. Validation: buyer needs sufficient cash; seller needs shares. No short-selling, no margin
  5. Settlement: atomic transfer of cash and shares

Per-Round Dynamics

FV(t) = (T − t) × E[d] (DLM 2005 declining FV)
divt ∈ {0, 2·E[d]} with Pr = 0.5
cashi,t+1 = cashi,t + divt × sharesi,t

Each period: (1) update beliefs, (2) compute reservation orders, (3) CDA match, (4) pay dividends to share-holders, (5) record VWAP, volume, and best bid/ask.

Risk Composition Effects

The simulator reveals how agent population composition drives bubble formation:

  • High risk-loving (γ<0): narrow spreads, aggressive bidding → prices overshoot FV → large NAPD and amplitude.
  • Balanced (RL ≈ RN ≈ RA): prices oscillate around the declining FV path; smaller bubbles.
  • High risk-averse (γ>0): wide spreads, conservative bids → prices undershoot FV → thin trade, low turnover.

Stochastic vs LLM agents

The same DLM protocol runs whether the agent uses the closed-form CARA reservation rule or a real LLM call (Claude / GPT). With no API key the CARA stochastic model is used, giving fully reproducible runs. With an API key the LLM produces beliefs that are then plugged into the same CDA — enabling direct A/B comparison of bubble metrics under identical market conditions.

Main Result 2: Experience Effect

Two channels of "experience"

We model the DLM experience effect with the simplest possible primitive: every replay of the market multiplies the inexperienced agents’ bias, anchor, noise, and momentum parameters by 0.5. After two replays they retain a quarter of their initial bias; after four, only 1/16. This deliberately minimal model is enough to reproduce the qualitative pattern in DLM Table 2.

decay = 0.5experience — applied to bias, anchor, noise, momentum

Within-Session: α (transfer treatment)

The Experienced % slider sets the fraction α of agents who, from period 1, already track FV with only small noise. This is the DLM "transfer" treatment in pure form: how many experienced traders are needed before the bubble disappears? Runs at α = 0, 1/3, 1/2, 2/3, 1 give a within-session counterpart to DLM §II.

Across-Session: Experience Replays

The Experience Sessions slider runs additional replays. Same agents, reset endowments, persistent experience counter. The Experience Effect chart (Fig. 6) plots Haessel-R², NAPD, and Amplitude across sessions — the direct in-silico analogue of DLM Table 2.

Expected Outcome

Across replays: R² rises monotonically, NAPD and Amplitude fall. With sufficient experience, the realized price hugs the declining FV path almost perfectly — bubbles diminish, exactly as DLM (2005) reported for human subjects.

Conclusions & Future Directions

Summary of Contributions and Open Research Questions

Key Contributions

  1. Naive in-silico replacement of human subjects in the DLM (2005, AER) bubble experiment with Lopez-Lira (2025) CARA / LLM trading agents. The market structure (declining FV, stochastic dividends, CDA) is preserved exactly.
  2. Two orthogonal experience levers: within-session α (DLM transfer treatment) and across-session experience replays (DLM main result), both reproduced from a single set of agents with no manual parameter tuning.
  3. Hybrid agent backend: closed-form CARA stochastic model by default, with optional LLM (Claude / GPT) drop-in via API key. Same protocol, two decision engines, direct A/B comparison.
  4. Open-source, no-build SPA deployable to GitHub Pages. Fully interactive sidebar parameters, six real-time Plotly charts, Canvas 2D Trading Floor visualization, JSON / CSV export, EN/ZH i18n.

Main Findings (qualitative)

  • Inexperienced-only CARA / LLM markets reproduce DLM-style bubbles: low Haessel-R², high NAPD and amplitude.
  • Replaying the market with the same agents rapidly suppresses the bubble — R² rises, NAPD falls, matching the DLM Table 2 pattern.
  • Within a single session, increasing the experienced fraction α produces the same qualitative effect at much lower cost.
  • Risk-loving (γ<0) populations inflate bubbles further than risk-averse (γ>0) populations.

Future Directions

  • Real LLM agents at scale: run the same experiment with Claude, GPT-5, and open-weight models; quantify how each model maps onto the DLM experience curve.
  • Mixed-experience markets: directly recreate DLM Round 4 by sprinkling fresh inexperienced LLM agents into a previously trained pool.
  • Belief-formation ablations: test alternative learning rules (Bayesian updating, RL) instead of the simple bias-decay primitive.
  • Mechanism design: swap the CDA for call auction or frequent batch auction and measure the bubble effect.
  • Human / AI hybrid markets: place real human traders alongside LLM agents and study emergent interaction dynamics.

Lopez-Lira (2025) §2; DLM (2005) fn.5

Declining Fundamental Value

FV(t) = (T − t) × E[d]. The asset's intrinsic value declines linearly as remaining dividend periods decrease, reaching zero at period T.

FV(t) = (T − t) × E[d]
DLM (2005) §I declining-FV asset

Reservation Prices

CARA-optimal bid/ask prices incorporating a risk premium from market uncertainty σ.

bid = belief − γσ²/2  |  ask = belief + γσ²/2
Lopez-Lira (2025) §2.2

Bubble Formation

Occurs when the market price persistently exceeds fundamental value, driven by inexperienced agents' optimism bias and momentum chasing.

P(t) > FV(t) for many periods → positive NAPD
DLM (2005) Table 2; bubble metrics

Experience Effect

DLM (2005) found bubbles diminish when traders replay the market. We model this by halving the residual bias of inexperienced agents at each replay.

decay = 0.5experience
DLM (2005) §II Table 2 p.1735

Continuous Double Auction (CDA)

Market mechanism where agents submit limit orders. A trade executes whenever the highest bid meets or exceeds the lowest ask.

match while max(bid) ≥ min(ask); P = (bid+ask)/2
Smith (1962); Lopez-Lira (2025) §3
